## できること

- 1 クリックでスクリーンショットを保存・クリップボードへコピー
- 非アクティブ状態での音声再生
- ウィンドウを最前面で固定
- アカウントのログアウト（Cookie の削除）

## 注意事項

- **課金周りについては動作検証を行っておりません**
- シャイニーカラーズ以外のサイトを表示した際の動作については確認しておりません **（enza を含む）**
